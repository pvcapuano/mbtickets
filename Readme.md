## Resumo: 

Aplicativo para compra de ingressos para eventos e palestras feito em Reaact Native.


## Rodando o projeto:

* git clone https://github.com/pvcapuano/mbtickets.git
* cd mbtickets
* npm i (instala todas as dependências)
* expo start (inicia o projeto)
